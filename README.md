## 🧐 Funkcjonalności

Oto kilka z najlepszych funkcji projektu:

*   Wybranie miasta z wyszukiwarki
*   Wybranie miasta z mapy
*   Informacje o pogodzie
*   Prognoza na następne 5 dni
*   Wybrane miasto jest zapisywane
*   Ciemny i jasny motyw

  
  
## 💻 Języki i API

W projekcie użyłem:

*   React
*   Openweathermap 2.5
